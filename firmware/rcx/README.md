# RCX Firmware

`firm0332.lgo` is the LEGO Mindstorms RCX 2.0 firmware image (~2001), originally
distributed by LEGO as part of the RCX 2.0 retail kit. The RCX hardware has
been discontinued for over two decades and the firmware has been redistributed
unmodified by the LEGO robotics community — BrickOS, NQC, leJOS, and countless
fan projects — since its original release. LEGO has never released the
firmware under an open-source license and, to the best of our knowledge, has
never objected to community redistribution for interoperability with
end-of-life hardware.

BrickLogo includes this file solely to load onto end-of-life RCX hardware that
LEGO no longer supports. The copyright remains with the LEGO Group, and this
inclusion should not be construed as a license. If the LEGO Group has any
objection to its redistribution here, please open an issue on the BrickLogo
repository and it will be removed promptly.
